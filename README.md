# Aegis Demo

A recruiter-friendly preview of the Aegis project, designed to showcase system capabilities and architecture without exposing the full source code.

## Purpose
This repository is a **public-facing demo** that demonstrates key platform capabilities using mock data and static UI components. Perfect for:
- **Hiring portfolios** - Showcase your technical skills and project vision to recruiters
- **Team sharing** - Let collaborators see project capabilities without full access
- **Stakeholder presentations** - Demonstrate system architecture and features
- **Open source contributions** - Share your work with the developer community

## What it shows
- Realtime sensor mesh visibility
- Operational status metrics
- Active alert notifications
- High-level system capability cards and architecture summary
- **Code samples**: Simplified examples of AI processing, sensor data handling, and gRPC integration
- Recruiter-friendly product capability showcase

## What it does not include
- Full `backend/` microservices
- `blockchain/` smart contract code or deployment tooling
- `database/` schemas, migrations, or production data
- Secure configuration, authentication, or production integration code

## Code Samples
The demo includes simplified, anonymized code snippets that demonstrate:
- Acoustic pest recognition algorithms
- Sensor data aggregation and health monitoring
- gRPC service communication patterns

These samples are educational examples only and do not contain proprietary algorithms or production-ready implementations.

- All code in this repository is owned by the Aegis project creator
- The full Aegis codebase remains private and proprietary

## How to run
1. Open `demo_app/index.html` in a browser.
2. The demo uses mock data and is ready to view without any server.

## Static run
1. ```cd c:\Users\Mahantesh\DevelopmentProjects\AegisDemo\demo_app```
2. ```npx serve .```  or   ```python -m http.server 8080```
